This project is NOT a generic website.

It is a cinematic ceremonial digital environment.

The website must feel:
- immersive
- spacious
- sacred
- architectural
- emotionally deep
- restrained
- intentional

NOT:
- startup
- SaaS
- wellness app
- generic luxury brand
- coaching website
- retreat marketplace
- spiritual cliché
